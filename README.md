# landleg-node

地腿 Node.js 版本

**需要 Node.js 环境: [https://nodejs.org/zh-cn/download/](https://nodejs.org/zh-cn/download/)**

## 安装 & 使用

### 安装
```
$ npm install landleg -g
```

### 使用
```
# 默认，第一次使用需要填写用户名密码
$ landleg 

# 填写用户名密码并登录
$ landleg --login 
OR
$ landleg -i

# 注销
$ landleg --logout
OR
$ landleg -o
```
